Rubric Builder & Scorer API

An API for generating evaluation rubrics, rubric refinement, and scoring text outputs. Powered by LLMs.

Interactive docs: /docs

Endpoints

POST /api/generate-rubric — Generate 5 scoring dimensions from a prompt
POST /api/refine-rubric — Validate user-edited dimensions
POST /api/score — Score text against a rubric
POST /api/rubrics — Save a rubric
GET /api/rubrics — List saved rubrics
GET /api/rubrics/{id} — Get a rubric by ID
DELETE /api/rubrics/{id} — Delete a rubric

Sample Commands

1. Generate a Rubric

curl -s -X POST {{ base_url }}/api/generate-rubric \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Is building language models a profitable business?"}'

2. Refine a Rubric

curl -s -X POST {{ base_url }}/api/refine-rubric \
  -H "Content-Type: application/json" \
  -d '{
    "rubric": [
      {"name": "Clarity", "description": "How clearly the idea is expressed"},
      {"name": "Depth", "description": "How thoroughly the topic is explored"},
      {"name": "Accuracy", "description": "Factual correctness of the content"},
      {"name": "Creativity", "description": "Originality and novelty of the argument"},
      {"name": "Evidence", "description": "Quality and relevance of supporting evidence"}
    ]
  }'

3. Score Text Against a Rubric

curl -s -X POST {{ base_url }}/api/score \
  -H "Content-Type: application/json" \
  -d '{
    "input_text": "Climate change is a pressing issue caused by greenhouse gas emissions.",
    "rubric": [
      {"name": "Clarity", "description": "How clearly the idea is expressed"},
      {"name": "Depth", "description": "How thoroughly the topic is explored"},
      {"name": "Accuracy", "description": "Factual correctness of the content"},
      {"name": "Relevance", "description": "How relevant the content is to the topic"},
      {"name": "Conciseness", "description": "How efficiently the point is made"}
    ]
  }'

4. Save a Rubric

curl -s -X POST {{ base_url }}/api/rubrics \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Writing Quality",
    "rubric": [
      {"name": "Clarity", "description": "How clearly the idea is expressed"},
      {"name": "Depth", "description": "How thoroughly the topic is explored"}
    ]
  }'

5. List Saved Rubrics

curl -s {{ base_url }}/api/rubrics

6. Get a Rubric by ID

curl -s {{ base_url }}/api/rubrics/YOUR_RUBRIC_ID

7. Delete a Rubric

curl -s -X DELETE {{ base_url }}/api/rubrics/YOUR_RUBRIC_ID

Typical Flow

  1. Generate — provide a prompt, get 5 evaluation dimensions
  2. Refine — edit dimensions as needed, validate via the refine endpoint
  3. Score — submit text to be scored against the rubric
  4. Save — persist the rubric for future use
